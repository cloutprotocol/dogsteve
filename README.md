## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Deploy to Vercel

```bash
npm run build
```

Then deploy to Vercel or run:

```bash
vercel deploy
```

## Customization

- Update contract address in `app/page.tsx`
- Add your social links in the same file
- Modify logo text in `app/components/Nintendo64Logo.tsx`
